Welcome to WebStarter

A browser is a software application for accessing information on the World Wide Web.
Each individual web page, image, and video is identified by a distinct URL, enabling browsers to retrieve and display them on the user's device.
(wikipedia)
You are not here to learn HTML, CSS or JavaScript.
So what!
You are here to take a look at how these technologies are connected to each other in the browser...
(me ;-)

The browser engine is capable of interpreting an HTML document to build the DOM (Document Object Model), that is to say the web page.
The DOM can be then accessed and manipulated by CSS and JavaScript technologies through an API made available by the browser.
